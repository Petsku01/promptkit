# SCAMPER Brainstorming

**Category:** creative
**Source:** Creative thinking framework

## When to Use

Generating novel features, product ideas, or marketing angles.

## The Prompt

```
I want to improve my product: [Product description].

Use the SCAMPER framework to generate 7 distinct ideas:
- Substitute: [idea]
- Combine: [idea]
- Adapt: [idea]
- Modify/Magnify: [idea]
- Purpose (put to another use): [idea]
- Eliminate: [idea]
- Rearrange/Reverse: [idea]
```

## How It Works

Applying a rigid, proven creative framework prevents the LLM from outputting generic, bland brainstorm lists by forcing it to view the problem from 7 specific, distinct angles.
