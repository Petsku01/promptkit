## Tulkintaohje

**Pisteet ovat suuntaa-antavia** - eri benchmarkit mittaavat eri asioita.

Most important:
1. Testaa omalla for your use case
2. Use your own data for evaluation
3. Benchmarkit vanhenevat nopeasti
