# o1 / o1-mini Prompting Guide

## Model Characteristics
- **Type:** Reasoning model (thinks before responding)
- **Context:** 128K (o1), 128K (o1-mini)
- **Strengths:** Complex reasoning, math, code, logic
- **Best for:** Hard problems requiring deep thinking

## Key Difference

o1 models think internally - DON'T ask them to "think step by step"!

## ❌ Wrong (Don't Do This)

```
Think step by step and solve this math problem.
Show your reasoning.
```

## ✅ Correct

```
Solve this math problem.
```

The model reasons internally. Just state the task.

## Best Practices

```
# Simple and direct
Solve: What is the integral of x^2 * e^x?

# For code
Write a function that finds the longest palindromic substring.

# For analysis  
Analyze the time complexity of this algorithm.
```

## When to Use o1 vs GPT-4

| Task | Use |
|------|-----|
| Complex math | o1 |
| Multi-step logic | o1 |
| PhD-level problems | o1 |
| Creative writing | GPT-4 |
| Simple Q&A | GPT-4o |
| Speed critical | GPT-4o |

## Key Tips

- Don't say "think step by step" - it thinks automatically
- Don't ask for reasoning - it's internal
- Keep prompts simple and direct
- More expensive + slower, use for hard problems only
- o1-mini is faster/cheaper for simpler reasoning tasks
