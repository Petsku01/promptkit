# GPT-4o: Writing

## Tekstin parannus

```
Improve this text:

Goals:
- Fix grammar and spelling
- Improve clarity
- Make it more concise
- Maintain original tone

Provide:
1. Improved version
2. List of changes made

Text:
```

## Summary

```
Summarize this text:

Length: [X sentences / X words]
Focus: [key points / main argument / action items]
Audience: [technical / general / executive]

Text:
```

## Email writing

```
Write an email:

Purpose: [request / follow-up / announcement / etc]
Tone: [formal / friendly / urgent]
Recipient: [colleague / client / manager]
Key points to include:
- [point 1]
- [point 2]

Keep it concise.
```

## Dokumentaatio

```
Write documentation for:

[paste code/API/feature]

Include:
- Overview (what it does)
- Usage examples
- Parameters/options
- Return values
- Common errors
```

## Translation + localization

```
Translate and localize:

Source: [language]
Target: [language]
Context: [formal business / casual / technical]

Adapt idioms and cultural references appropriately.

Text:
```

## Copywriting

```
Write [type of content]:

Product/topic: [description]
Audience: [target audience]
Goal: [inform / persuade / entertain]
Tone: [professional / playful / urgent]
Length: [X words]
CTA: [desired action]
```
