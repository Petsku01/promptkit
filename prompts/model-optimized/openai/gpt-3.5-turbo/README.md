# GPT-3.5 Turbo

## Characteristics
- **Context:** 16K tokens
- **Strengths:** Fast, cheap, reliable
- **Best for:** Simple tasks, high volume

## Prompt Tips

- Keep prompts shorter than GPT-4
- Be more explicit with instructions
- Use examples (few-shot) for complex tasks
- Simpler outputs work better

## Use Cases

- Classification
- Simple extraction
- Basic Q&A
- Text formatting
- Quick completions

## Limitations

- Less reasoning capability
- Shorter context
- May need more guidance

See GPT-4o prompts and simplify for this model.
