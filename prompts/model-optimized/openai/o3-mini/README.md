# o3-mini

## Characteristics
- **Type:** Reasoning model (smaller)
- **Strengths:** Math, logic, coding
- **Best for:** Complex reasoning at lower cost

## Key Rules

 **DO NOT use "think step by step"** - counterproductive for reasoning models

## Prompt Style

```
Solve this problem:

[problem]

Provide your answer.
```

The model handles reasoning internally - don't over-prompt.

## Use Cases

- Math problems
- Logic puzzles
- Code debugging
- Multi-step reasoning

## Compared to o1/o3

- Faster, cheaper
- Less deep reasoning
- Good for simpler reasoning tasks
