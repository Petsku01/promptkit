## Milloin käyttää agenttista promptia?

- Tehtävä vaatii **useita vaiheita**
- Tarvitaan **ulkoisia työkaluja** (haku, laskenta, API)
- Itsekorjaus ja **reflektio** tarpeen
- Monimutkainen **päätöksenteko**
