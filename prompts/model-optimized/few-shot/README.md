## Few-Shot -periaatteet

1. **Count:** 3-5 examples usually sufficient
2. **Monipuolisuus:** Kata eri tapaukset
3. **Format:** Input/Output clearly separated
4. **Johdonmukaisuus:** Sama formaatti kaikissa
5. **Edge cases:** Including edge cases
