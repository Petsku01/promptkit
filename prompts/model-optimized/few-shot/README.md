## Few-Shot -periaatteet

1. **Määrä:** 3-5 esimerkkiä yleensä riittää
2. **Monipuolisuus:** Kata eri tapaukset
3. **Formaatti:** Input/Output selkeästi erotettu
4. **Johdonmukaisuus:** Sama formaatti kaikissa
5. **Edge cases:** Mukana myös erikoistapauksia
