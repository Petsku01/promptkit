# Qwen (Alibaba) Models

## Mallit

| Malli | Koot | Paras käyttö | Promptit |
|-------|------|--------------|----------|
| [Qwen 2.5](qwen-2.5/) | 0.5B - 72B | Yleiskäyttö, koodi | 5+ |
| [Qwen 3](qwen-3/) | 0.6B - 235B | MoE, reasoning | 5+ |
| [Qwen 3.5](qwen-3.5/) | 0.6B - 32B | Nopea, edge | 5+ |

## Avainominaisuudet

- **Monikielinen**: Erinomainen kiina + englanti
- **Pitkä konteksti**: Jopa 128K tokenia (isommat)
- **Koodi**: Qwen-Coder variantit
- **MoE**: Qwen 3 mixture-of-experts

## Yleiset vinkit

- Toimii hyvin yksinkertaisilla prompteilla
- Strukturoitu output: anna esimerkki
- Few-shot parantaa johdonmukaisuutta
- Pienemmät mallit: lyhyemmät promptit
