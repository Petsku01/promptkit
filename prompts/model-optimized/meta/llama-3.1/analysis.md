# Llama 3.1: Analysis

## Chain-of-Thought Reasoning

```
Problem: [problem]

Think through this step by step:

1. First, understand what's being asked
2. Identify the key information
3. Plan your approach
4. Execute step by step
5. Verify your answer

Show all your work.
```

## Document Analysis

```
Analyze this document:

[document text]

Provide:
1. **Summary** (3-5 sentences)
2. **Key Points** (bullet list)
3. **Main Conclusions**
4. **Questions Raised**
5. **Action Items** (if any)
```

## Research Summary

```
Topic: [topic]

Based on your knowledge:

1. **Overview** - What is this?
2. **Key Concepts** - Main ideas
3. **Current State** - Where things stand
4. **Controversies** - Debates/disagreements
5. **Future Outlook** - What's next

Note: My knowledge has a cutoff date. For current events, verify externally.
```

## Pro/Con Analysis

```
Evaluate: [topic/decision]

**PROS:**
1. [pro + explanation]
2. [pro + explanation]
3. [pro + explanation]

**CONS:**
1. [con + explanation]
2. [con + explanation]
3. [con + explanation]

**VERDICT:** [recommendation with reasoning]
```

## Llama-Specific Tips

**For 8B model:**
- Keep prompts shorter
- One task at a time
- Clearer structure

**For 70B+ model:**
- Can handle complex reasoning
- Better at nuanced analysis
- More reliable CoT
