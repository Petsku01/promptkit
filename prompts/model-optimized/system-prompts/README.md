Kuka olet, mitä osaat

[KÄYTTÄYTYMINEN]
Miten vastaat, tyyli

[RAJOITUKSET]
Mitä et tee

[OUTPUT-FORMAATTI]
Miten muotoilet vastaukset

[ESIMERKIT] (valinnainen)
