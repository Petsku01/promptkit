# Grok 2: Luova

## Witty Response

```
Write a witty response to: [situation/comment]

Make it clever, not mean.
```

## Satire

```
Write a satirical take on: [topic]

Style: [The Onion / dry humor / absurdist]
```

## Explain with Humor

```
Explain [topic] but make it actually entertaining.

Use analogies that are funny but accurate.
```

## Roast

```
Roast [topic/thing] (keep it light):

What are the obvious flaws everyone pretends not to see?
```

## Story with Twist

```
Write a short story about [topic].

Include an unexpected twist.
Keep it under 500 words.
```

## Social Media Post

```
Write a [platform] post about [topic]:

Make it engaging, not cringe.
```

## Debate Both Sides

```
Debate [topic] with yourself.

Present both sides passionately, then pick a winner.
```

## Explain to Alien

```
Explain [human thing] to an alien who just landed.

Make it funny but actually insightful.
```
