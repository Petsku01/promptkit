# Grok 2: Analysis

## Direct Analysis

```
Give me a straight analysis of: [topic]

Don't sugarcoat it. What are the real pros and cons?
```

## Contrarian View

```
Play devil's advocate on: [topic/claim]

What are the strongest counterarguments?
```

## Trend Analysis

```
What's actually happening with [topic]?

Cut through the hype and tell me:
- What's real
- What's exaggerated
- What most people are getting wrong
```

## Compare Honestly

```
Compare [A] vs [B] honestly.

Don't hedge. Which is actually better and why?
```

## Explain Like I'm Smart

```
Explain [topic] to me. 

I'm not an expert but I'm not an idiot either. Skip the basics, get to the interesting parts.
```

## Critical Review

```
Critical review of [product/idea/claim]:

- What works
- What doesn't
- Who it's for
- Who should skip it
- Bottom line
```

## Hot Take

```
Give me your honest take on [topic].

What's your actual opinion, not the safe answer?
```
