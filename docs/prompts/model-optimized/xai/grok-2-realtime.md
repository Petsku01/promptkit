# Grok 2: Real-time information

## Current Events

```
What's happening with [topic] right now?

What are people saying about it?
```

## Trending

```
What's trending about [topic]?

What's the actual discourse vs media spin?
```

## Latest Updates

```
Give me the latest on [topic/event/company].

Focus on what happened in the last [time period].
```

## Public Opinion

```
What do people actually think about [topic]?

Not what media says, what regular people are saying.
```

## Breaking Down News

```
Break down this news: [topic]

- What happened
- Why it matters
- What people are missing
- What happens next
```

## Sentiment Check

```
What's the sentiment around [company/person/topic] right now?

Positive, negative, or mixed? Why?
```

## Compare Coverage

```
How is [topic] being covered differently by different sources?

What's the spin from each side?
```

## Timeline

```
Give me a timeline of [recent event].

What happened in order?
```
