# Grok 3: Reasoning

## Real-time Analysis
```
Analyze the current state of [topic].

Use your real-time information access to provide:
1. Latest developments
2. Key trends
3. Expert opinions
4. Data-backed insights

Be direct and don't hedge unnecessarily.
```

## Complex Problem Solving
```
Problem: [describe problem]

Think through this step by step:
1. What are the core constraints?
2. What approaches could work?
3. What are the tradeoffs?
4. What's your recommended solution?

Be bold in your recommendations.
```

## Contrarian Analysis
```
Topic: [topic]

Give me the contrarian take on this. Challenge conventional wisdom.

What do most people get wrong about this, and why?
```

---
**Grok 3 Tips:**
- Grok has real-time access to X/Twitter data
- It's designed to be less restricted and more direct
- Can handle controversial topics without excessive hedging
- Good at current events and real-time analysis
