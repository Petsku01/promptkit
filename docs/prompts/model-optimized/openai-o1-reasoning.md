# O1: Deep Reasoning

## Complex Problem (Let O1 Think)
```
Solve this problem. Take your time to reason through it carefully.

[describe complex problem]
```

Note: O1 works best with minimal prompting. State the problem clearly and let it reason.

## Math/Logic Problem
```
[State the problem directly]

Show your work.
```

## Code Architecture
```
Design a system that:
- [requirement 1]
- [requirement 2]
- [requirement 3]

Consider tradeoffs and justify your choices.
```

## Research Question
```
[State your research question]

Analyze from multiple angles and provide a well-reasoned answer.
```

## Strategy Problem
```
[Describe the strategic situation]

What is the optimal approach and why?
```

---

**O1 Tips:**
- Keep prompts short and clear
- Don't over-explain or add excessive structure
- Let the model do the reasoning work
- Works best for math, code, logic, and strategy
