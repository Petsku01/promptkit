# Claude Opus 4.5: Extended Thinking

## Complex Analysis with Extended Thinking
```xml
<task>
[Describe complex task requiring deep analysis]
</task>

<context>
[Provide relevant background and data]
</context>

<instructions>
This requires careful analysis. Use extended thinking to:
1. Map out all relevant factors
2. Consider multiple perspectives
3. Identify potential blind spots
4. Reason through implications
5. Arrive at a well-justified conclusion
</instructions>

<output_format>
Provide:
- Your reasoning process
- Key insights discovered
- Final recommendation with confidence level
</output_format>
```

## Strategic Decision Making
```xml
<decision_context>
[Describe the strategic decision]
</decision_context>

<stakeholders>
[List key stakeholders and their interests]
</stakeholders>

<constraints>
[List constraints and requirements]
</constraints>

<request>
Analyze this decision thoroughly. Consider:
- Short and long-term implications
- Risk factors and mitigations
- Alternative approaches
- Second-order effects

Provide a recommended path forward with clear reasoning.
</request>
```

## Research Synthesis
```xml
<sources>
[Paste multiple sources or documents]
</sources>

<synthesis_task>
Synthesize these sources into a coherent understanding:
1. Identify areas of consensus
2. Map contradictions and their underlying causes
3. Evaluate methodology and evidence quality
4. Note gaps in collective knowledge
5. Provide integrated conclusions with confidence levels
</synthesis_task>
```

---
**Claude Opus 4.5 Tips (from Anthropic docs):**
- Best for complex reasoning and analysis tasks
- Use XML tags to structure complex prompts
- Extended thinking mode enables deeper reasoning
- Explicitly request reasoning transparency
- Higher cost - use for high-value analysis
- Excellent at nuanced, multi-perspective analysis
