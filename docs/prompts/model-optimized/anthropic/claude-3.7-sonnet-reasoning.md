# Claude 3.7 Sonnet: Hybrid Reasoning

## Extended Thinking Activation
```xml
<task>
[Describe complex task]
</task>

<instructions>
This is a complex problem. You should take time to think through it step-by-step using extended thinking before providing your final answer.
1. Map out the constraints.
2. Outline possible approaches.
3. Validate your logic before answering.
</instructions>
```

## Architectural Design
```xml
<task>
Design a highly scalable architecture for [system].
</task>

<requirements>
[List requirements]
</requirements>

<evaluation_criteria>
- Scalability
- Security
- Maintainability
</evaluation_criteria>
```

---
**Claude 3.7 Sonnet Tips:**
- It is a hybrid model that can reason extensively when prompted.
- Use XML tags to structure the context and instructions clearly.
- Ask it to explicitly plan and validate before writing the final output.
