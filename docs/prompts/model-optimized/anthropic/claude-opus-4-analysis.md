# Claude Opus 4: Deep Analysis

## Comprehensive Research Analysis
```xml
<task>
Conduct a thorough analysis of [topic].
</task>

<analysis_framework>
1. **Current State**: What is the situation now?
2. **Historical Context**: How did we get here?
3. **Key Players**: Who are the main stakeholders?
4. **Competing Perspectives**: What are the different viewpoints?
5. **Evidence Quality**: How reliable is the available information?
6. **Implications**: What are the consequences of various outcomes?
7. **Gaps**: What don't we know?
</analysis_framework>

<output_requirements>
- Cite specific sources and data points
- Acknowledge uncertainty explicitly
- Distinguish between facts, inferences, and speculation
- Provide actionable conclusions
</output_requirements>
```

## Strategic Decision Analysis
```xml
<decision>
[Describe the decision to be made]
</decision>

<context>
[Provide relevant background]
</context>

<constraints>
[List constraints and requirements]
</constraints>

<analysis_request>
Analyze this decision using:
1. Option generation (at least 5 distinct approaches)
2. Criteria weighting
3. Risk assessment for each option
4. Second-order effects
5. Recommended path with confidence level
</analysis_request>
```

## Document Synthesis
```xml
<documents>
[Paste multiple documents or summaries]
</documents>

<synthesis_task>
Create a unified understanding by:
1. Identifying common themes
2. Mapping contradictions
3. Evaluating source reliability
4. Filling inferential gaps
5. Producing integrated conclusions
</synthesis_task>
```

---
**Claude Opus 4 Tips:**
- Opus excels at deep, nuanced analysis
- Use XML structure for complex multi-part tasks
- Request explicit uncertainty acknowledgment
- Best for tasks requiring careful reasoning over quick responses
- Higher cost - use for high-value analysis tasks
