# Codestral: Specialized Coding

## Code Generation
```
[language] function that [does what]:

Requirements:
- [req 1]
- [req 2]
```

## Fill in the Middle (FIM)
```
Complete the code between the markers:

[code before]
<FILL>
[code after]
```

## Code Completion
```
Continue this code:

[paste partial code]
```

## Quick Fix
```
Fix this code:

[paste buggy code]

Error: [error message if any]
```

## Test Generation
```
Write tests for this function:

[paste function]

Use [pytest/jest/etc].
```

---

**Codestral Tips:**
- Optimized specifically for code
- Great at fill-in-the-middle tasks
- Fast code completion
- Best for pure coding tasks, not conversation
