# Agentic: Reflection

## Self-Critique

```
I just completed: [action/output]

Critique my work:

1. **Correctness**: Is this actually right?
2. **Completeness**: Did I miss anything?
3. **Quality**: Could this be better?
4. **Assumptions**: What did I assume that might be wrong?

If issues found, fix them before finalizing.
```

## Iterative Refinement

```
First attempt at: [task]

[initial output]

---

Reflection round 1:
- What's good: [strengths]
- What's weak: [problems]
- Specific fixes: [improvements]

Refined version:
[improved output]

---

Reflection round 2:
[repeat until satisfied or max 3 rounds]
```

## Error Analysis

```
Something went wrong: [error/failure]

Analyze:
1. What exactly failed?
2. Why did it fail? (root cause)
3. What should I have done differently?
4. How to fix it now?
5. How to prevent this next time?

Apply the fix and verify.
```

## Output Verification

```
My answer/output: [output]

Verification checklist:
□ Does it answer the actual question?
□ Are all facts verifiable?
□ Is the logic sound?
□ Are there any contradictions?
□ Would an expert agree?

Issues found: [list]
Corrections: [fixes]

Final verified output:
```

## Confidence Assessment

```
My response: [response]

Rate my confidence:
- Factual accuracy: [1-10] because [reason]
- Completeness: [1-10] because [reason]
- Reasoning quality: [1-10] because [reason]

Overall confidence: [high/medium/low]

If low confidence areas exist, what would increase confidence?
```

## Learning from Feedback

```
I received this feedback: [feedback]

Analysis:
1. Is the feedback valid? [yes/no + reasoning]
2. What specifically should change?
3. Does this apply to similar future tasks?

Updated approach:
[improved version based on feedback]

Lesson learned:
[key takeaway for future]
```

## Meta-Cognition

```
Task I'm working on: [task]

Before continuing, reflect:

1. **Understanding**: Do I fully understand what's being asked?
2. **Approach**: Is my current approach optimal?
3. **Progress**: Am I making progress or stuck?
4. **Alternatives**: Are there better ways to do this?
5. **Blind spots**: What might I be missing?

Based on reflection: [continue/adjust/ask for help]
```
