# Claude Opus: Deep Analysis

## Comprehensive Analysis
```xml
<task>
Perform a thorough, multi-dimensional analysis of the following.
</task>

<analysis_framework>
1. **Surface Level**: What is explicitly stated
2. **Underlying Patterns**: Implicit themes and structures
3. **Context**: Historical, cultural, or domain-specific factors
4. **Contradictions**: Internal tensions or inconsistencies
5. **Implications**: Second and third-order consequences
6. **Blind Spots**: What is notably absent or unaddressed
</analysis_framework>

<material>
[paste content]
</material>

<output_requirements>
- Support claims with specific references
- Acknowledge uncertainty where present
- Provide actionable insights where applicable
</output_requirements>
```

## Strategic Reasoning
```xml
<task>
Analyze this strategic situation with full depth.
</task>

<situation>
[describe situation]
</situation>

<analysis_dimensions>
1. **Stakeholder Mapping**: All parties and their interests
2. **Power Dynamics**: Who has leverage and why
3. **Constraints**: Hard limits vs. negotiable boundaries
4. **Time Horizons**: Short vs. long-term considerations
5. **Option Space**: Full range of possible actions
6. **Second-Order Effects**: How each option changes the game
7. **Risk Assessment**: What could go wrong and probability
8. **Recommendation**: Best path forward with reasoning
</analysis_dimensions>
```

## Research Synthesis
```xml
<task>
Synthesize these sources into a coherent understanding.
</task>

<sources>
[paste multiple sources]
</sources>

<synthesis_requirements>
- Identify areas of consensus
- Map disagreements and their underlying reasons
- Evaluate methodology quality of each source
- Note gaps in collective knowledge
- Provide integrated conclusion
- Rate confidence level (high/medium/low) with justification
</synthesis_requirements>
```
