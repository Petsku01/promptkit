# Gemini 2.5: Reasoning

## Long Document Analysis
```
Analyze this long document:

[paste document - Gemini can handle up to 2M tokens]

Provide:
1. Executive summary
2. Key arguments/points
3. Supporting evidence quality
4. Logical gaps or weaknesses
5. Actionable conclusions
```

## Multi-source Synthesis
```
I have these sources on [topic]:

Source 1: [content]
Source 2: [content]
Source 3: [content]

Synthesize into:
1. Points of agreement
2. Points of disagreement
3. Unique contributions from each
4. Integrated conclusion
5. Remaining questions
```

## Research Question Analysis
```
Research question: [your question]

Background: [context]

Help me:
1. Refine the question
2. Identify sub-questions
3. Suggest methodology
4. Anticipate challenges
5. Propose success criteria
```

---
**Gemini 2.5 Reasoning Tips:**
- Massive context window - use it for large documents
- Good at maintaining coherence over long inputs
- Strong at multi-source analysis
- Can reference specific parts of long documents accurately
