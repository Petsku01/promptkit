# Gemini 2.5 Pro: Reasoning

## Deep Analysis
```
Analyze this thoroughly:

[topic/document]

Cover:
1. Background context
2. Key components and relationships
3. Underlying assumptions
4. Logical structure
5. Implications and consequences
6. Recommendations

Show your reasoning for each section.
```

## Multi-Source Synthesis
```
I have these sources on [topic]:

Source 1: [content]
Source 2: [content]
Source 3: [content]

Synthesize into:
1. Points of agreement
2. Points of disagreement
3. Unique contributions from each source
4. Integrated conclusion
5. Remaining questions and gaps
```

## Research Question Analysis
```
Research question: [your question]

Background: [context]

Help me:
1. Refine the question
2. Identify sub-questions
3. Suggest methodology
4. Anticipate challenges
5. Propose success criteria
6. Identify relevant frameworks
```

## Critical Evaluation
```
Critically evaluate:

[claim/argument/proposal]

Analysis:
1. Core thesis
2. Strengths
3. Weaknesses
4. Missing considerations
5. Alternative perspectives
6. Verdict with confidence level
```

## Step-by-Step Problem Solving
```
Solve this problem step by step:

[problem]

Show your work:
1. Understand: What is being asked?
2. Identify: Key information and constraints
3. Plan: Break into logical steps
4. Execute: Work through each step
5. Verify: Check your reasoning
6. Conclude: State your answer

Explain your reasoning at each step.
```

## Decision Framework
```
Help me decide between:

Option A: [description]
Option B: [description]
Option C: [description]

Criteria:
1. [criterion 1]
2. [criterion 2]
3. [criterion 3]

For each option, evaluate against each criterion.
Then recommend with justification.
```

---

**Gemini 2.5 Pro Reasoning Tips:**
- Pro excels at multi-step reasoning — don't oversimplify
- Use "show your reasoning" or "think step by step" for deeper analysis
- Structured output sections help Pro organize complex responses
- 1M context allows analyzing multiple long documents simultaneously
- For best results, specify what depth of analysis you expect
