# GPT-5: Advanced Reasoning

## Complex Problem Solving
```
Problem: [describe problem]

Requirements:
- [requirement 1]
- [requirement 2]

Approach this systematically:
1. Clarify any ambiguities
2. Break down into sub-problems
3. Solve each component
4. Integrate into final solution
5. Verify correctness
```

## Multi-step Analysis
```
Task: [describe analysis task]

Data/Context:
[provide data]

Perform the following analysis:
1. Initial observations
2. Pattern identification
3. Hypothesis generation
4. Testing hypotheses against data
5. Conclusions with confidence levels
```

## Code Architecture Design
```
Design a system for: [description]

Constraints:
- Scale: [expected scale]
- Performance: [requirements]
- Budget: [constraints]

Provide:
1. High-level architecture diagram (in text/mermaid)
2. Component breakdown
3. Data flow
4. Technology recommendations
5. Potential bottlenecks and mitigations
```

---
**GPT-5 Tips:**
- Strong at multi-step reasoning
- Can maintain context over long conversations
- Good at code and technical tasks
- Works well with structured prompts
- Can generate and interpret diagrams (mermaid, ASCII)
