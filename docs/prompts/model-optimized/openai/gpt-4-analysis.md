```
Analyze the following using this structure:

**Overview**: 2-3 sentence summary
**Key Points**: Bullet list of main elements
**Strengths**: What works well
**Weaknesses**: Areas of concern
**Recommendations**: Actionable next steps

Content:
[paste content]
```
