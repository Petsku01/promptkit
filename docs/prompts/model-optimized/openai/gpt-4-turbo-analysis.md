# GPT-4 Turbo: Analysis (Long Context)

## Large Document Analysis
```
Analyze this long document. I need:

1. Executive Summary (200 words max)
2. Key themes identified
3. Important quotes with context
4. Questions the document raises
5. Gaps or missing information

Document:
[paste long document - up to 128k tokens]
```

## Multi-Document Synthesis
```
Synthesize these multiple documents:

Document 1: [title]
[content]

Document 2: [title]
[content]

Document 3: [title]
[content]

Provide:
- Common threads across all documents
- Contradictions between them
- Unique insights from each
- Overall synthesis
```

## Conversation/Thread Analysis
```
Analyze this conversation thread:

[paste long conversation]

Identify:
1. Main topics discussed
2. Key decisions made
3. Action items mentioned
4. Unresolved questions
5. Sentiment/tone shifts
```
