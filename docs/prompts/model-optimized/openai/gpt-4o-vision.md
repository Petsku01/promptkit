# GPT-4o: Visionn (Image Analysis)

## General Image Analysis

```
Analyze this image thoroughly:

1. **Objects**: What objects are present? List them.
2. **Text**: Is there any text? Transcribe it exactly.
3. **People**: Describe any people (count, actions, positions)
4. **Setting**: Where is this? (indoor/outdoor, location type)
5. **Context**: What is happening in this image?
6. **Details**: Note any interesting or unusual details.

Be specific and detailed.
```

## OCR / Text Extraction

```
Extract all text from this image.

Format:
- Preserve original layout where possible
- Note any text that is unclear with [unclear]
- Separate distinct text blocks with ---

Return only the extracted text.
```

## Image Comparison

```
Compare these two images:

1. What is similar?
2. What is different?
3. Which is [better/clearer/more relevant] and why?

Be specific about locations of differences.
```

## Diagram Interpretation

```
Analyze this diagram/chart:

1. **Type**: What kind of diagram is this?
2. **Data**: What data/information does it show?
3. **Trends**: What patterns or trends are visible?
4. **Key insights**: What are the main takeaways?
5. **Anomalies**: Anything unexpected?
```

## UI/Screenshot Analysis

```
Analyze this screenshot:

1. **App/Website**: What application or website is this?
2. **Current state**: What screen/page is shown?
3. **UI elements**: List key buttons, menus, inputs visible
4. **User action**: What action appears to be in progress?
5. **Issues**: Any UX problems visible?
```
