# Defensive Prompts

Prompts for security, safety, and adversarial defense.

## Browsing Prompts

```bash
# List all providers and models
promptkit prompts

# List defensive prompts
promptkit prompts --model defensive

# Search for a specific prompt
promptkit search "code review"
```
