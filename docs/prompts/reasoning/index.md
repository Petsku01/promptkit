# Reasoning Prompts

Prompts for reasoning, analysis, and critical thinking.

## Browsing Prompts

```bash
# List all providers and models
promptkit prompts

# List reasoning prompts
promptkit prompts --model reasoning

# Search for a specific prompt
promptkit search "code review"
```
