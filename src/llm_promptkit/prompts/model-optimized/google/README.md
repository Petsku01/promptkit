# Google (Gemini)

## Models

| Model | Best for | Prompts |
|-------|--------------|----------|
| [Gemini Pro](gemini-pro/) | General purpose | 5+ |
| [Gemini Flash](gemini-flash/) | Speed, volume | 5+ |
| [Gemini 2](gemini-2/) | Latest, multimodal | 5+ |
| [Gemini 2.5](gemini-2.5/) | Reasoning, long context | 2 |
| [Gemini 2.5 Flash](gemini-2.5-flash/) | Fast, efficient, 1M context | 3 |
| [Gemini 2.5 Pro](gemini-2.5-pro/) | Advanced reasoning, multimodal | 3 |

## Strengths

- Multimodal (image, video, audio)
- Long context (up to 2M tokens)
- Good grounding
- Fast (Flash models)

## General Tips

- Structured output works well
- Bullet points and lists
- Clear sections
