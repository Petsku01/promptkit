# Grok 3 Mini: Reasoning

## Quick Analysis (Fast Mode)
```
Analyze this quickly:

[describe topic or paste content]

Give me:
1. Key takeaway
2. Main risk / downside
3. What most people miss

Be direct. No fluff.
```

## Step-by-Step Reasoning (Thinking Mode)
```
Think through this step by step.

Problem: [describe problem]
Constraints: [any constraints]

1. What are we actually trying to solve?
2. What are the possible approaches?
3. What are the tradeoffs of each?
4. Which is best and why?

Be bold in your recommendations. Don't hedge unless genuinely uncertain.
```

## Real-Time Topic Analysis
```
What's happening with [topic] right now?

Use your real-time data to give me:
1. Latest developments
2. What the conversation looks like on X
3. Emerging trends
4. Unpopular takes worth considering

Don't just summarize — tell me what matters.
```

## Contrarian Analysis
```
Everyone says [conventional wisdom about topic].

Give me the contrarian take:
- What are people getting wrong?
- What's the strongest counterargument?
- If the consensus is wrong, what does that imply?

Be honest, not edgy for the sake of it.
```

## Cost-Effective Research
```
I need to understand [topic] at a high level.

Give me:
1. The 30-second summary
2. Key concepts I need to know
3. Where the real complexity hides
4. What I should read next if I want depth

Skip the basics I can find on Wikipedia. Get to the interesting parts.
```

---

**Grok 3 Mini Reasoning Tips:**
- Much cheaper than full Grok 3 — use for volume reasoning tasks
- Enable thinking mode for harder problems (still very cost-efficient)
- Real-time X/Twitter data available — leverage for current events
- Direct style works best — no need for elaborate system prompts
- Good for quick iterations — run many queries cheaply
- Not meant for deeply complex reasoning (use full Grok 3 for that)
