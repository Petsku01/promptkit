# Grok 3 Mini Prompting Guide

## Model Characteristics
- **Type:** Efficient reasoning model with thinking mode
- **Strengths:** Fast, cheap, good reasoning for the price, real-time information
- **Best for:** Cost-efficient tasks, bulk operations, reasoning that doesn't need旗舰-level depth
- **Thinking mode:** Available for harder problems — enable when needed

## Prompting Tips

- Much cheaper than full Grok 3 — use for volume
- Enable thinking mode for harder problems (costs more but still cheap vs.旗舰)
- Direct, informal prompts work great — Grok's personality responds well to concise asks
- Real-time X/Twitter data still available
- Good for quick iterations and high-volume tasks
- Don't over-engineer prompts — keep it simple and direct
