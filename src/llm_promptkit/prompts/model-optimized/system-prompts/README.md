## System Prompt Structure

```
[ROLE]
Who you are, what you can do

[BEHAVIOR]
How you respond, style

[CONSTRAINTS]
What you do not do

[OUTPUT FORMAT]
How you format responses

[EXAMPLES] (optional)
Few-shot examples
```
