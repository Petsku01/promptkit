# Meta (Llama) Models

## Models

| Model | Best For | Prompts |
|-------|----------|---------|
| [Llama 3.1](llama-3.1/) | Long context, tool use | 5 |
| [Llama 3.2](llama-3.2/) | Visionn, edge deployment | 5 |
| [Llama 3.3](llama-3.3/) | Best instruction following | 9 |

## Key Differences

- **Llama 3.1**: 128K context, tool calling, 8B/70B/405B
- **Llama 3.2**: Multimodal (vision), edge-optimized, 1B-90B
- **Llama 3.3**: 70B matches 405B quality, best overall

## General Tips

- Simple, direct prompts work best
- Show exact output format you want
- System prompt for persona
- Open weights - good for fine-tuning
