## Document Analysis
```
Analyze this document for: [purpose]

Document: [paste text or describe]

Extract:
1. Key claims and their supporting evidence
2. Logical structure of the argument
3. Factual assertions (verifiable vs. speculative)
4. Tone and persuasive techniques
5. Missing information or logical gaps

Rate overall: evidence quality / logical coherence / completeness (1-10 each)
```

## Technical Architecture Review
```
Review this system architecture:

Architecture: [describe or paste diagram]
Requirements: [what it needs to support]

Evaluate:
1. **Scalability**: Can it handle 10x growth?
2. **Reliability**: Single points of failure?
3. **Security**: Attack surface and mitigations?
4. **Cost**: Resource efficiency
5. **Maintainability**: Complexity and coupling
6. **Extensibility**: Can new features be added easily?

Prioritized recommendations:
- Critical (fix now)
- Important (next sprint)
- Nice-to-have (backlog)
```
