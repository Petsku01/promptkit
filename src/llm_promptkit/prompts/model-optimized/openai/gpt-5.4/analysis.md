# GPT-5.4: Analysis

## Comprehensive System Analysis
```
Analyze this system/service/process:

System: [name and description]
Purpose: [what it's supposed to do]
Current state: [performance metrics, issues, etc.]

Analyze across dimensions:
1. **Efficiency**: Resource usage vs. output
2. **Reliability**: Uptime, error rates, failure modes
3. **Scalability**: Can it handle growth?
4. **Security**: Attack surface and vulnerabilities
5. **Compliance**: Regulatory requirements met?

For each dimension:
- Current score (1-10) with justification
- Top 3 improvement opportunities
- Quick wins vs. long-term investments

Overall assessment and prioritized roadmap.
```

## Data-Driven Decision Support
```
Analyze this data to inform a decision:

Question: [what decision needs to be made]
Data available: [describe datasets, metrics, or paste data]
Context: [business/technical context]

Steps:
1. Clean and summarize the data
2. Identify key trends and patterns
3. Calculate relevant statistics
4. Test for significance where applicable
5. Visualize key relationships (describe chart types)
6. List findings with confidence intervals
7. Make evidence-based recommendation

Note: If data is insufficient, state what additional data is needed and why.
```

## Incident Post-Mortem
```
Write a post-mortem for this incident:

Incident: [brief description]
Duration: [start time - end time]
Impact: [users/systems/data affected]
Severity: [P1/P2/P3]

Sections:
1. **Summary** (2-3 sentences)
2. **Timeline** (UTC, with timestamps)
3. **Root Cause** (technical explanation)
4. **Impact Assessment** (quantified)
5. **What Went Wrong** (failures in process/tech)
6. **What Went Right** (what mitigated the damage)
7. **Action Items** (with owners and deadlines)

Action items must be SMART: Specific, Measurable, Achievable, Relevant, Time-bound.
```

---
**GPT-5.4 Analysis Tips:**
- Provide raw data when possible — it handles quantitative analysis well
- Ask for explicit confidence levels on conclusions
- Can process and cross-reference multiple document sources
- Benefits from comparison frameworks and rubrics
- Use structured output formats for reproducible analysis
