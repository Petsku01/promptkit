# o3: Reasoning

## Math Problem
```
Solve: ∫₀¹ x³ · ln(x) dx

Give the final answer in simplified form.
```

## Logic Puzzle
```
Five houses in a row are painted different colors. Each house is occupied by a person of different nationality. Each person drinks a different beverage, smokes a different brand, and keeps a different pet.

[State the clues]

Who owns the fish?
```

## Algorithm Design
```
Design an algorithm for [problem description].

Constraints:
- Time: O(n log n) maximum
- Space: O(n) maximum
- Must handle edge cases: [list them]

Provide:
1. The algorithm (pseudocode or real code)
2. Time complexity analysis
3. Space complexity analysis
```

## Scientific Analysis
```
Analyze this experimental data:

[paste data]

Determine:
1. The underlying relationship between variables
2. Statistical significance
3. Potential sources of error
4. Conclusions that can be drawn
```

## Mathematical Proof
```
Prove that [mathematical statement].

State any assumptions or axioms used.
```

## Research Question
```
[State your research question]

Analyze from multiple angles and provide a well-reasoned answer.
```

## Complex Decision Analysis
```
Decision: [what needs to be decided]
Options: [A, B, C]
Constraints: [budget, time, technical, etc.]

For each option, evaluate expected outcomes. Recommend the best option with justification.
```

---
**o3 Reasoning Tips:**
- Keep prompts short and clear — the model does the reasoning internally
- Don't over-explain or add excessive structure
- State the problem, provide constraints, let it work
- Harder problems produce better results — save o3 for genuinely difficult tasks
- Provide all necessary information upfront — it won't ask clarifying questions mid-reasoning
- Specify desired output format if you need structured responses
