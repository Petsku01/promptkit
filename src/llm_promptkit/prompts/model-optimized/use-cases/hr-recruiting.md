# Use Case: HR & Recruiting

## Resume Screener

```
Screen this resume for the position:

Position: [job title]
Required skills: [must-have skills]
Preferred skills: [nice-to-have]
Experience level: [years]

Resume:
[resume text]

Screening:
- Required skills match: [X/Y matched]
- Preferred skills: [list found]
- Experience level: [meets/exceeds/below]
- Red flags: [gaps, inconsistencies]
- Highlights: [standout qualifications]

Recommendation: [proceed/maybe/reject]
Fit score: [1-10]
```

## Job Description Generator

```
Create a job description:

Role: [title]
Team: [which team]
Level: [junior/mid/senior/lead]
Key responsibilities: [main duties]
Must-haves: [required skills]
Nice-to-haves: [preferred skills]

Generate:
## [Title]

### About the Role
[exciting summary]

### What You'll Do
- [responsibility 1]
- [responsibility 2]
...

### What You'll Bring
- [requirement 1]
- [requirement 2]
...

### Bonus Points
- [nice-to-have]
...

### Why Join Us
[company pitch]
```

## Candidate Feedback Writer

```
Write interview feedback:

Candidate: [name]
Position: [role]
Interviewer notes: [raw notes]

Structured feedback:
## Summary
[2-3 sentence overview]

## Strengths
- [strength + evidence]

## Areas of Concern
- [concern + evidence]

## Technical Assessment
[1-5] - [justification]

## Cultural Fit
[1-5] - [justification]

## Recommendation
[Strong hire / Hire / Maybe / No]

## Suggested Next Steps
[what to explore in next round]
```
