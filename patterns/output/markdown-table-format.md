# Markdown Table Format Pattern

## Description

Request output in a structured markdown table format for easy reading and data processing.

## When to use

When comparing items, listing structured data, or creating reference tables.

## When to avoid

Complex nested data that doesn't fit a flat table structure.

## Template

```
[Task description]

Output your response as a markdown table with the following columns:
| Column 1 | Column 2 | Column 3 |

Include a header row and separator line.
```

## Example

```
Compare the top 5 JavaScript frameworks for web development.

Output your response as a markdown table with the following columns:
| Framework | Best For | Learning Curve | Community Size |

Include a header row and separator line.

Example row format:
| React | Complex SPAs | Moderate | Very Large |
```
