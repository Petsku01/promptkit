# Step-by-Step Analysis Pattern

## Description

Break down complex analysis into sequential steps with clear reasoning at each stage.

## When to use

For complex decisions, root cause analysis, or debugging where multiple factors interact.

## When to avoid

Simple yes/no questions or straightforward classifications.

## Template

```
[Complex problem or decision]

Analyze this step-by-step:
1. **Problem Definition**: Clarify what we're trying to solve
2. **Data Gathering**: List relevant facts and observations
3. **Hypothesis Generation**: Brainstorm possible explanations/causes
4. **Analysis**: Evaluate each hypothesis against the data
5. **Conclusion**: Synthesize findings into actionable insights
6. **Recommendation**: Propose next steps with rationale

Show your work at each step.
```

## Example

```
Our e-commerce site had a 40% drop in conversions last week. The drop started on Tuesday. No code deployments were made. Marketing spend remained constant.

Analyze this step-by-step:
1. **Problem Definition**: Clarify what we're trying to solve
2. **Data Gathering**: List relevant facts and observations
3. **Hypothesis Generation**: Brainstorm possible explanations
4. **Analysis**: Evaluate each hypothesis
5. **Conclusion**: Synthesize findings
6. **Recommendation**: Propose next steps

Show your work at each step.
```
